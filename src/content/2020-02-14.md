유명 프로그래머의 개인적인 최고 VSCode extensions 10개를 소개하겠습니다.

10번째 extension 말고는 VSCode extension market 탭에서 각 제목을 검색하면 찾으실 수 있습니다.

### 1. ESLint

ESLint 는 자바스크립트 코드에 에러가 있으면 표시해주는 도구입니다.
자동으로 linting configuration 을 읽어줘서 문제점들을 알려주고 심지어 해결해 주는 경우도 있습니다.

### 2. GitLens

여러분 다들 git 사용하시죠? git 을 통한 협업을 더욱 효율적으로 할 수 있게 도와줍니다. blame, code authorship, activity heatmaps, recent changes, file history 그리고 commit search 등의 git 도구들이 내장되어 있습니다.

### 3. Debugger for Chrome

디버그 중요하죠. Chrome 이나 Chromium 에서 자바스크립트 코드를 디버깅 해줄 수 있게 해줍니다. 일부의 기능만 말하자면 breakpoints, callstack 검사, 함수 내부 보기 등이 있습니다.

### 4. Bracket Pair Colorizer 2

일치하는 괄호들을 같은 색으로 표시해줍니다. 코드를 더 빨리 읽을 수 있도록 해주고 결과적으로 성능을 향상시켜줍니다.

### 5. Bookmarks

중요한 위치를 저장하고 탐색하여 빠르고 쉽게 다시 돌아가게 해줌으로써 파일과 파일 사이를 왔다 갔다 하는 시간을 줄여줄 것입니다.

### 6. TODO Highlight

남은 할일들을 TODO 주석 리스트를 쫙 뽑고, 배경을 눈에 돋보이게 해줘서 간단하고 편리하게 해줍니다.

### 7. Live Server

간편하게 로컬호스트 서버를 열 수 있게 해줘서 미리보기와 디버깅이 쉬워지고, 무엇보다 실시간 새로고침이 된다.

### 8. Rest Client

VSCode 에서 HTTP 요청과 응답을 볼 수 있게 해줍니다. 넓은 범위의 formats 와 authorization 을 지원해주고, 거의 대부분의 설정에 작동될 것입니다.

### 9. One Dark Pro

개발자들이 정말로 사랑하는 테마입니다. 깔끔하며 예쁜 섹채로 대조 효과를 주며 마음이 편해집니다. 감성 코딩 ~ ㅋㅋㅋ

### 10. Fira Code

<a href="https://github.com/tonsky/FiraCode" target="_blank">Fira Code</a>
는 VSCode extension 이 아니여서 셋업하는데 추가적인 절차가 필요할 수 있지만, 적응되면 개간지 프로그래밍 폰트, 합자를 통해 코드를 더 빨리 읽을 수 있습니다.
