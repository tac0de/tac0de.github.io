Krontium
============

Astro 기반 장편 소설 연재 정적 사이트입니다. 작품명은 `Krontium`이며, 신의 뼈에서 나온 금속과 기억세를 중심으로 한 실험적 UI/UX 포트폴리오입니다.

루트 페이지는 브라우저 언어를 감지해 `/ko/` 또는 `/en/`으로 이동합니다.

## 개발

```sh
npm install
npm run dev
```

## 검증

```sh
npm run build
npm run verify
```

## 회차 추가

`src/content/episodes`에 Markdown 파일을 추가하면 연재 목록과 상세 페이지에 자동 반영됩니다.
