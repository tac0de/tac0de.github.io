JavaScript Iterator 란 무엇이며 어디에서 사용할 수 있는가?

15 March, 2020 - 2 min read

JavaScript iterators 는 ES6에 도입되었으며, 일련의 값들, 보통 모음에 사용됩니다.

정의에 따르면, iterators 는 next() 함수를 구현해야 하며, 여기서 값은 {value, done} 형태로 객체를 변환하는 반복 순서의 다음 값이고 done은 시퀀스가 이미 소비되었는지 여부를 결정합니다.

실제 프로젝트에서 실용적으로 사용하는 매우 단순한 iterators 는 다음과 같습니다.

lass LinkedList {
  constructor(data) {
    this.data = data;
  }

  firstItem() {
    return this.data.find(i => i.head);
  }

  findById(id) {
    return this.data.find(i => i.id === id);
  }

  [Symbol.iterator]() {
    let item = {next: this.firstItem().id};
    return {
      next: () => {
        item = this.findById(item.next);
        if(item) {
          return {value: item.value, done: false};
        }
        return {value: undefined, done: true};
      }
    };
  }
}

const myList = new LinkedList([
  {id: 'a10', value: 'First', next: 'a13', head: true },
  {id: 'a11', value: 'Last', next: null, head: false },
  {id: 'a12', value: 'Third', next: 'a11', head: false },
  {id: 'a13', value: 'Second', next: 'a12', head: false }
]);

for(let item of myList) {
  console.log(item);    // 'First', 'Second', 'Third', 'Last'
}

위의 예에서, 우리는 LinkedList 데이터 구조를 구현하는데, 각 항목이 value 가 있는 data 배열을 내부적으로 사용하며, 순서의 위치를 결정하는 데 사용되는 구현별 속성도 함께 사용합니다.

이 클래스에서 생성된 객체는 기본적으로 반복될 수 없는 것이 아니기 때문에, 우리는 Symbol.iterator 를 사용하여 반복기를 정의하고 반환된 시퀀스가 클래스의 내부 구현에 기초하여 순서대로 정렬되도록 설정하며, 반환된 품목은 그 value만 반환합니다.

관련 참고에서, iterators는 기능일 뿐, 다른 함수(예: 기존 iterators 에게 반복을 위임하는 것)로 불릴 수 있는 동시에, Symbol.iterator 이름에 제한되지 않아 동일한 개체에 대해 복수의 반복자를 정의할 수 있습니다.

여기 이러한 개념의 예가 있습니다.

class SpecialList {
  constructor(data) {
    this.data = data
  }

  [Symbol.iterator]() {
    return this.data[Symbol.iterator]()
  }

  values() {
    return this.data
      .filter(i => i.complete)
      .map(i => i.value)
      [Symbol.iterator]()
  }
}

const myList = new SpecialList([
  { complete: true, value: "Lorem ipsum" },
  { complete: true, value: "dolor sit amet" },
  { complete: false },
  { complete: true, value: "adipiscing elit" },
])

for (let item of myList) {
  console.log(item)
}

for (let item of myList.values()) {
  console.log(item) // 'Lorem ipsum', 'dolor sit amet', 'adipiscing elit'
}

이 예제에서는 data 개체의 네이티브 배열 iterators 를 사용하여 SpecialList 를 쉽게 사용할 수 있게 하여 data 배열의 정확한 값을 반환합니다.

한편, Array.protype 을 사용하여 iterators 그 자체인 values 방법도 정의합니다.

data 배열의 filter() 및 Array.protype.map() 를 반환한 다음, 결과의 Symbol.iterator 를 반환하고, 시퀀스가 비어 있지 않은 객체에서만 반복을 허용하고 각 객체에 대한 값만 반환합니다.
