CSS Variables 란 무엇이고 언제 써야할까?

15 March, 2020 - 2 min read

CSS 변수(공식적으로 CSS 사용자 지정 속성)는 다른 언어의 변수와 매우 비슷하게 동작하므로, CSS 문서 내에서 재사용할 특정 값을 포함하는 명명된 변수를 정의할 수 있습니다.

이들은 항상 두 개의 대시(예: --my-color: 검정;)로 시작하고 var() 함수(예: 색상:--my-color)를 사용하여 액세스하는 사용자 지정 속성 표기법을 사용하여 정의됩니다.

사용자 지정 특성은 다른 요소와 구성요소(예: 수직 리듬, 인쇄 변수, 색상 팔레트 등) 간에 스타일을 공유하는 데 특히 유용합니다.

CSS 변수 사용의 가장 일반적인 예 중 하나는 테마와 다크 모드인데, 여기에서 CSS 변수는 전체 웹사이트에 공유 팔레트를 만든 다음, 공통 조상(예: <body> 요소 또는 최상위 컨테이너)에 클래스를 적용하여 다른 것과 쉽게 교환할 수 있습니다.

이 예는 최상위 수준의 조상이 :root 요소로부터 값을 상속받으므로 :root 요소와 계단식(cascading)에 정의된 전역 변수를 입증하는 데 도움이 됩니다.

/* 글로벌 변수는 :root 에서 정의됩니다. */
:root {
  --bg-color: #fff;
  --main-color: #000;
  --secondary-color: #222;
}
/* 부모로부터 변수를 상속 받습니다. */
body {
  background-color: var(--bg-color);
  color: var(--main-color);
}
small {
  color: var(--secondary-color);
}
/* 새롭게 변수와 값들을 정의하며, 상속된 것을 무시할 수 있습니다. */
body.dark {
  --bg-color: #080808;
  --main-color: #fff;
  --secondary-color: #ccc;
}

또 다른 다소 유용한 예는 특정 요소의 특정 변형에 대해 공유된 사용자 정의 스타일을 정의하여 스타일을 반복하지 않고도 전체 구성요소 트리를 사용자 지정할 수 있도록 하는 것입니다.

이 예제는 이전 예보다 계단식 구성을 훨씬 더 잘 보여주며, 또한 다른 요소들 간에 스타일을 공유하는 아이디어를 소개합니다.

.btn {
  --bg-color: #002299;
  --text-color: #fff;
  --highlight-color: #669900;

  background-color: var(--bg-color);
  color: var(--text-color);
}
/* --highlight-color 는 .btn 의 자식에도 쓰일 수 있습니다. */
.btn .highlight {
  color: var(--highlight-color);
}
/* .btn.danger .highlight 는 btn-danger 에 정의된 --highlight-color 를 쓸것입니다 . */
.btn-danger {
  --bg-color: #dd4a68;
  --text-color: #000;
  --highlight-color: #990055;
}

마지막으로 CSS 변수 작업에 대한 다음과 같은 유용한 팁을 기억합시다.

  • 두 번째 인수를 var() 함수에 제공하여(예: var(--text-color, black) --text-color 가 정의되지 않은 경우) 폴백 값을 정의할 수 있습니다.
  • CSS 변수는 대소문자를 구분하므로 대문자에 유의하십시오. 그것들은 또한 다른 스타일처럼 HTML로 밑줄칠 수 있습니다(예: <div style="--text-color: red").
  • 다른 변수를 폴백으로 사용하여 var() 호를 중첩할 수 있으며(예: val(-main-color, var(-other-color))), 이를 calc()와 같은 다른 함수에 전달하거나(예: --text-color: var(---main-color)), 한 변수를 다른 함수에 할당할 수도 있습니다.
